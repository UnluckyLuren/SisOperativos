# Webpack Starter

Proyecto inicial para crear paginas o aplicaciones con webpack

### Notas:

Recordar el construir los modulos de node con el comando:

```
    npm install
```

Y para construir los builds

Development:
```
    npm run build:dev
```

Production:
```
    npm run build
```
